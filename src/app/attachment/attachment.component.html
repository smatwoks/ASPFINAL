    Data processing device Chromatography Data Station JC2000

    Data Processing Device Chromatography Data Station This is a data processing device for the JC2000 gas chromatograph. It allows easy selection and operation of functions such as chromatographic analysis, saving and playing back chromatographic data, enlarging and reducing chromatographic data, various quantitative calculations, and controlling external devices. It also supports various optional functions such as measurement result tables and automatic PDF saving. As standard, it can process data independently for two channels, and can also process and print two channels simultaneously. It also allows easy reprocessing on the screen by loading saved chromatograms while capturing data from two channels. (An optional four-channel version is also available.)

    Data processing device Chromatography Data Station JSμ7

    This is a data processing device for gas chromatographs. It allows easy selection and operation of functions such as chromatographic analysis, saving and replaying chromatographic data, enlarging and reducing chromatographic data, various quantitative calculations, and control of external devices (optional). Low cost has been achieved by limiting the functions to the bare minimum. In addition, a simple calibration function has been added, allowing calibration at the touch of a button. Software for controlling external devices is optional.

    D/A converter DAC-201

    DAC-201 DA converter Outputs a signal in the set range

  • Features

    • This device receives the measurement results sent from the gas chromatograph data processing device via RS232C, displays the concentration of each component on the LCD screen, and outputs a 4-20mA (DC) signal within the set range.

    Flame extinction detector GFA-201

    Flame extinction detector GFA-201

  • Features

    • This device detects the extinction of the hydrogen flame of the FID detector. It detects the extinction of the flame with a temperature sensor and outputs an alarm and a detection signal. In addition, if you install the optional hydrogen shutoff unit, it will automatically shut off hydrogen when the flame goes out, preventing it from diffusing into the room. (If you purchase it at the same time as a GC, the hydrogen shutoff valve is built into the GC.)
